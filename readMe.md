# ASP.NET Core Application - Interactive studying calendar
# <sub>The following technologies are used in this application:<sub>
1. ASP.NET Core v3.1
2. Entity Framework Core
3. AngularJS v1.8.0
4. HTML5/CSS3/JavaScript
5. JQuery v3.5.1
6. BootStrap v4.5.2
7. MS SQL Server 2019
